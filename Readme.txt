A simple project on web development that works both on front-end and back-end as well.

1.The input (username,password,email,profile_pic) is taken from the users and stored in the database.

2.Their are certain conditions for the inputs to be valid and different from others off-course.

3.In the database we can play with the inputs and make their status change using a drop-down.

4. Drop-down consist of 4 options that is placed,shipped,drawn and return.

5.You can either individually visit the users and change their status or you can select 
them all from list of users and change their status accordingly,
